# owo

Like tree, but outputs your entire codebase into a single markdown file - perfect for LLM context (if you cant afford claude code &lt;/3)

## Installation

### Manual Installation

```bash
mkdir -p ~/.owo
git clone https://github.com/waterfallorg/owo.git ~/.owo
cd ~/.owo
cargo install --path .
```

### Direct Cargo Install

If you have the source code locally:

```bash
cargo install --path .
```
